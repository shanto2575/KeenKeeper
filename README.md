## Project Name
KeenKeeper

## Short Description

KeenKeeper is a personal relationship management web app that helps users keep track of meaningful friendships and stay connected regularly.
It allows users to monitor interactions (Call, Text, Video), track relationship goals, and view analytics of their communication habits.

The goal of this project is to make sure no important relationship gets forgotten 


### Technologies Used

## Frontend
-React JS
-React Router
-Tailwind CSS
-DaisyUI

## State Management
-React Context API
-Hooks

## Data Visualization
Recharts

### Key Features
1️⃣ Friendship Interaction Tracker

Users can quickly log interactions with friends:
📞 Call
💬 Text
🎥 Video

All interactions are stored and shown in timeline & analytics.

2️⃣ Friendship Analytics Dashboard

Visual charts help users understand their relationship habits:

Pie chart of interaction types
Monthly interaction stats
On-track vs Need-attention friends

3️⃣ Relationship Goal Reminder System

Each friend has:

Contact frequency goal
Last contact tracking
Next due date reminder
